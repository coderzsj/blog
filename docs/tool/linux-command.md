通常情况下，RPM 包采用系统默认的安装路径，所有安装文件会按照类别分散安装到表 1 所示的目录中。

RPM 包默认安装路径

| 安装路径            | 含义            |
|-----------------|---------------|
| /etc/           | 配置文件安装目录      |
| /usr/bin/       | 可执行的命令文件安装目录  |
| /usr/lib/       | 程序所使用的函数库保存位置 |
| /usr/share/doc/ | 基本的软件使用手册保存位置 |
| /usr/share/man/ | 帮助文件保存位置      |
